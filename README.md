# Análise de Faturamento E-Commerce (Olist DB) | End-to-End BI

Este é um projeto prático de Business Intelligence desenvolvido de ponta a ponta (end-to-end), transformando dados brutos de vendas em um painel estratégico para tomada de decisão comercial.

## Tecnologias Utilizadas
* *SQL (T-SQL):* Criação, estruturação e relacionamento das tabelas do banco de dados.
* *Power Query:* Processamento, limpeza e transformação de dados (ETL).
* *Power BI:* Modelagem de dados (Star Schema) e design de dashboard interativo.

## Estrutura do Projeto
1. *Modelagem de Dados (SQL):* Organização das tabelas através do script criacao_tables.sql para garantir a integridade dos relacionamentos.
2. *Tratamento & ETL:* Limpeza de categorias não informadas e padronização dos formatos de valores e moedas.
3. *Dashboard Executivo:* Desenvolvimento de um layout moderno focado em UX/UI (tema escuro) para monitoramento de métricas como Faturamento Total, Desempenho por Estado e Mix de Categorias de Produtos.

## Insights Gerados
* Identificação clara da concentração massiva do faturamento na região Sudeste (com SP e RJ liderando).
* Visualização dinâmica do ranking de categorias de produtos que mais trazem receita para a operação.
